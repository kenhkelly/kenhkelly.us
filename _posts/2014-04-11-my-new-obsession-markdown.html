Lately I’ve been really getting into Markdown, a text-to-HTML conversion tool for web writers. Markdown allows you to write using an easy-to-read, easy-to-write plain text format, then convert it to structurally valid XHTML (or HTML).

It’s great because it makes life so much easier when it comes to writing your content without worrying about formatting. I use it for my blog posts and it has greatly sped things up and kept everything in a similar format. There are a few tools out there that I’ve been using as well.

  • An extension I added to Firefox is called Markdown Here. Markdown Here will help you to write email more quickly and powerfully than you thought possible.
  • Tumblr supports Markdown, using a variation that has a few more features than the traditional Markdown - Called PHP Markdown
  • Draft also supports Markdown.

I strongly encourage you to read more about it and see how it’ll fit in to your flow.
