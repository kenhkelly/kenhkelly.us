Today I wanted to share a server uptime monitoring service I started to use. I’ve needed one of these for a while, but everything I found was paid. And then I found it - Uptime Robot.

Uptime Robot a pretty good free service. It checks the server in 5 minutes increments. If the server is down, it doesn’t just send you the alert immediately, it checks again to make sure it’s really down so not throw a false positive. It supports 50 different sites (which is plenty if you only use a few servers). It checks websites via the HTTP Port, but you can also set it up to check specific ports - I have it check my hosted Minecraft Server.

They offer an API, which is pretty interesting. I haven’t used it yet but it looks pretty straight-forward. You can receive alerts by any combination of email, sms, Twitter, or push notifications for iOS.

If this is a service you’ve been needing for your clients or yourself, I highly recommend giving it a look.
